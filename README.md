## ❓ What is this app?
MayMayLauncher is a quick launcher for Windows. 
Designed to quickly add pictures, videos, music, documents, games, and apps.
Best app for more space for your desktop or start menu instead of making folders and shortcuts.
Also, with a built-in media player!

## ❓ How to use this app?
At first, you will see a registration wizard
it is the profile creation wizard.
This wizard will help you to make your own local account.

```
Occasionally, 
you can visit again by going to Settings -> Launcher Tab -> Reset to Setup
or
Settings -> Launcher Tab -> Reset to Setup and Clear All Lists if you want to do a clean reset.
```

After you make your own local account,
Click **+ Add** or **Ctrl+"+"** to open your file, game, or app to add to the launcher tiles list.

```
Note: all files will be removed when a file is not found.
```

Double click on the item or select the item and press enter to open your file.

That's how you use it.
